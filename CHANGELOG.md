# Changelog

## [1.0.0] - 2026-05-31

### Added
- 扫描 6 个级别、85 周课程素材自动生成索引
- 单页应用：级别→周→天三级导航
- 图片讲义展示（支持缩放、轮播）
- 音频/视频内嵌播放器（自动播放下一个）
- PDF.js 内嵌阅读器（翻页、缩放）
- DOCX 歌词/译文内嵌显示
- 打卡功能 + 学习统计（天数、连续天数）
- 进度记忆（localStorage，自动恢复上次位置）
- iPad 局域网访问（响应式布局）
- 多线程 HTTP 服务器（单端口）
- 路径穿越防护、XSS 消毒
- macOS 一键启动脚本（start.command）
